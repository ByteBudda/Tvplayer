# IPTV Player

Современный и функциональный IPTV-плеер для Android, разработанный с использованием Kotlin и Jetpack Compose.

## Основные возможности
- **Загрузка плейлистов**: Поддержка форматов M3U.
- **Интеграция телепрограммы (EPG)**: Автоматическая загрузка телепрограммы для поддержки каналов.
- **Управление каналами**: Избранное, блокировка каналов (родительский контроль), редактирование.
- **Премиальный дизайн**: Современный интерфейс с использованием элементов Glassmorphism и поддержки темной/светлой тем.

## Технологический стек
- **Язык**: Kotlin
- **UI**: Jetpack Compose
- **База данных**: Room (локальное хранилище)
- **Сеть**: OkHttp

## Как использовать
1. Перейдите в раздел управления плейлистами и добавьте URL вашего M3U-файла.
2. Приложение автоматически импортирует каналы и попытается сопоставить их с доступными источниками EPG.
3. Используйте встроенные настройки для блокировки контента (родительский контроль) или добавления каналов в избранное.
